# 📘 바로인턴 직무과제 - Spring Boot API

Java Spring Boot 기반의 간단한 API 서버입니다. JWT 인증 및 Role 기반 접근 제어 기능이 포함되어 있으며, AWS EC2 인스턴스에서 배포 및 실행 중입니다.

## 🔗 프로젝트 개요

- **주요 기능**:
    - 회원가입 / 로그인
    - JWT 기반 인증
    - Role 기반 접근 제어
    - 관리자 권한 부여

- **기술 스택**:
    - Spring Boot
    - Spring Security
    - JWT
    - AWS EC2
    - Swagger
